#Go Style QR Code Reader mobile application

##How to start

#### 1) Install Node & ionic
Install [NodeJS](https://nodejs.org/en/download/).

in terminal:

    $ npm install -g @ionic/cli
  

#### 2) install project
open terminal in project folder

    $ npm install


#### 3) start project
open terminal in project folder

    $ ionic serve
    
open browser on http://localhost:8100
